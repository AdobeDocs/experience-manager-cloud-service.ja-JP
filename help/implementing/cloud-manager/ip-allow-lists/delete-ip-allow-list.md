# IP許可リストを削除しています{#delete-allow-list}

IP許可リストを削除するには、次の手順に従います。

1. 環境画面で、「ドメイン設定」ページに移動します。
1. 削除するカスタムドメイン名が一覧表示される行を指定します。
1. **を選択…**&#x200B;メニューを表示します。
1. 「削除」オプションを選択します。
1. 送信を確認します。

>[!NOTE]
>IP許可リストを削除すると、そのIPサーバーからすべてのサービスが自動的に適用解除され、テーブルから削除されます。
