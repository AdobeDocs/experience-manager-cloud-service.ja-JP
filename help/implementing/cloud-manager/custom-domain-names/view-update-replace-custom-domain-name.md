# カスタムドメイン名の表示、更新、置換 {#view-update-replace-cdn}

表示と更新メニューを使用すると、1 つまたは複数のカスタムドメイン名の詳細を表示できます。または、カスタムドメイン名に使用する SSL 証明書を変更できます。

>[!NOTE]
>SSL 証明書は有効で、カスタムドメイン名を含んでいる必要があります。
