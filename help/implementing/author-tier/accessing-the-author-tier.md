# オーサー層へのアクセス {#accessing-the-author-tier}

AEM as a Cloud Service 環境のオーサー層にアクセスするには、標準のカスタマイズできない URL を使用します。

現在、URL 形式は次のとおりです。

* `https://author-p[Program ID]-e[EnvironmentID].adobeaemcloud.com/`

AEMオーサー層のカスタムドメイン名はサポートされていません。
