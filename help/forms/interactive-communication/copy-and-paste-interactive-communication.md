# インタラクティブ通信エディターでのコピー&amp;ペースト

>[!NOTE]
>
> インタラクティブ通信機能は、アーリーアダプタープログラムで利用できます。 勤務先のアドレスから `aem-forms-ea@adobe.com` にメールを送信して、アクセスをリクエストします。

インタラクティブ通信（IC）エディターのコピー&amp;ペースト機能を使用すると、作成者は既存のインタラクティブ通信を複製して、別のフォルダーまたは場所で再利用できます。 この機能により、元のバージョンを変更することなく、ICの複数のバリエーションを有効にすることで、チームは効率的に作業できます。

コピーを作成することで、作成者はソースのインタラクティブ通信を維持しながら、様々なユースケース、顧客、ビジネスシナリオに合わせて安全に変更を加えることができます。

## コピーされる内容

インタラクティブ通信をコピーすると、次の要素が新しいICに複製されます。

- レイアウトと構造

- コンポーネントとフラグメント

- データのバインディングとルール

- スタイル設定とブランディング設定

- その後、コピーしたICでこれらの要素のいずれかを個別に変更できます。

## インタラクティブ通信のコピー

![IC ドキュメントを検索](/help/forms/interactive-communication/assets/copy-in-ic.png)

インタラクティブ通信をコピーするには、次の手順に従います。

- AEM インターフェイスでFormsとドキュメントに移動します。

- コピーするインタラクティブ通信を探します。

- ICを選択します。

- ツールバーまたはコンテキストメニューから、「コピー」を選択します。

- ターゲットのフォルダーまたは場所に移動します。

- 「ペースト」を選択して、インタラクティブ通信の複製を作成します。

- 新しいICは、元のICと同じ設定で作成されます。

## コピーしたインタラクティブ通信の編集と再利用

![IC ドキュメントを検索](/help/forms/interactive-communication/assets/paste-in-ic.png)

インタラクティブ通信をペーストした後：

- インタラクティブ通信エディターでコピーしたICを開きます。

- 必要に応じて、コンテンツ、ルール、データマッピング、レイアウトを更新します。

- 準備ができたら、ICを保存して公開します。

- コピーしたICに加えた変更は、元のインタラクティブ通信には影響しません。

## ベストプラクティス

- 元のICをベースラインまたはマスターバージョンとして保持します。

- コピーしたICの名前を明確に変更して、目的やバリエーションを反映します。

- コピー&amp;ペーストとバージョン管理およびコメントを組み合わせることで、トレーサビリティを向上させます。

- コピー後のデータバインディングと出力チャネルを確認して、正確性を確保します。

インタラクティブ通信のコピー&amp;ペースト機能は、作成者が既存のICを複製して個別に変更できるようにすることで、再利用とカスタマイズを簡素化します。 これにより、元のインタラクティブ通信に変更を加えるリスクを回避しながら、より迅速な開発、より安全な実験、一貫したコミュニケーション配信が可能になります。
