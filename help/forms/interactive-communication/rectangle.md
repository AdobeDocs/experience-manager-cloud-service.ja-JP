# インタラクティブ通信エディターの長方形コンポーネント


## &#x200B;1. はじめに

インタラクティブ通信（IC）エディターの長方形コンポーネントを使用すると、レイアウトディバイダ、視覚的なアクセント、コンテンツコンテナとして機能するシェイプグラフィカル要素を追加できます。長方形は、視覚的な階層を強化し、構造化されたコミュニケーションレイアウトでユーザーの注意を引きます。
このコンポーネントはデータと関連付けられていませんが、デザインの明確性を向上させ、関連するフィールドをグループ化し、全体的なプレゼンテーションを強化するのに役立ちます。

![IC ドキュメントを検索](/help/forms/interactive-communication/assets/rectangle.png)

## &#x200B;2. プロパティ

長方形コンポーネントには、カスタマイズ可能なプロパティがいくつか含まれています。

2.1. 名前

名前：長方形の一意の識別子。 主に、レイアウト階層で参照したり、一貫してスタイルを適用したりするために使用されます。

2.2. 位置

- **説明：** ドキュメントレイアウトで長方形が表示される場所を指定します。

- **設定：**

   - **XおよびY座標：**&#x200B;水平および垂直の配置を定義します。

   - **高さと幅（mm）:**&#x200B;長方形のサイズを制御します。

2.3. 余白

長方形コンポーネントの周囲の間隔を定義して、他のUI要素と分離します。

- 上（上）

- 下（下）

- 左

- 右

2.4. 外観

- **説明：**&#x200B;長方形の視覚的なスタイル設定を管理します。

- **カスタマイズ オプション：**

   - **塗りつぶし：**&#x200B;長方形の内部カラー。

   - **線：**&#x200B;長方形のアウトラインの色。

   - **線幅：**&#x200B;長方形の境界線の太さ。

   - **スタイル：** フラット、枠線、昇格などのプリセットスタイル。

   - **エッジ：** コーナーの外観（丸いエッジやシャープなエッジなど）を制御します。

2.5. プレゼンス

- **説明：**&#x200B;通信のレンダリング時に長方形を表示するかどうかを指定します。

- **オプション：**

   - **表示可能：**&#x200B;実行時に長方形を表示します。

   - **非表示：**&#x200B;長方形を表示せずにレイアウトスペースを維持します。

## &#x200B;3. 使用方法

長方形コンポーネントは、通常、コンテンツ入力ではなく、レイアウトやスタイル設定の目的で使用されます。 一般的なユースケースを次に示します。

- セクション間の視覚的な分離の作成

- グループ化された要素のハイライト

- 読みやすさと視覚的バランスの向上

- ヘッダー、フッター、コールアウトセクションの作成

長方形は、複雑なビジュアルデザイン構造を実現するために、サブフォームやコンテナなどの他のレイアウト要素と組み合わせることができます。

## &#x200B;4. ベストプラクティス

- レイアウト全体で長方形のスタイルを統一し、視覚的な調和を確保します。

- 隣接するフィールドが混雑するのを防ぐために、適切なマージンと間隔を適用します。

- ブランドやドキュメントのテーマに合った塗りと線の色を選択できます。

- 丸みを帯びたエッジを使用して、モダンなUI レイアウトのデザインを改善します。

- 長方形が編集中にデザイン目的でのみ必要で、最終的な出力では必要でない場合は、長方形を非表示にします。

長方形コンポーネントは、IC エディター内の非インタラクティブでありながら強力なツールです。 スタイルが設定され、効果的に配置されている場合は、データのバインディングやインタラクティブ性に複雑さが加わることなく、レイアウトの精度、視覚的なフロー、ユーザーエクスペリエンスが向上します。
