# インタラクティブ通信エディターでのデータバインディング


## &#x200B;1. はじめに

インタラクティブ通信エディターのデータバインディングは、キャンバス上のフィールドを管理されたデータレイヤーに接続し、コミュニケーションが実際のコンテキスト情報でレンダリングされるようにします。 コンポーネントをフォームデータモデル（FDM）にリンクすることで、制作者は正確性を確保し、手作業を削減して、動的にパーソナライズされたエクスペリエンスを提供できます。

ICのデータバインディングは、単に値を接続するだけでなく、ビジュアルマッピング、事前入力、同期をサポートしており、バックエンドシステムやデータモデルと連携しながら、より迅速にデザインすることができます。

![IC ドキュメントを検索](/help/forms/interactive-communication/assets/data-binding1.png)

## &#x200B;2. プロパティ

2.1 データ接続の管理（FDM）

- **FDMを選択：**&#x200B;適切なフォームデータモデル（顧客、アカウント、ポリシーなど）を選択します。 これにより、通信で使用されるフィールド、配列およびオブジェクトに対する権限のあるスキーマが確立されます。

- **データバインディングを作成：** バインディングを有効にすると、各フィールドをFDM パスに関連付けることができ、エラーを最小限に抑え、一貫した統合を確保できます。

- **データモデルへのフィールドのバインド：** ライブデータを使用したレンダリングを促進し、検証または条件付きロジックをサポートするために、特定のノード（customer.name、policy.holder.idなど）にフィールドをポイントします。

2.2 データバインディングの作成

- **ビジュアルマッピング：** フィールドとFDM ノード間のドラッグ&amp;ドロップ操作によるマッピングは、技術的な知識のないユーザーがミスを回避するのに役立ちます。

- **フィールドの関連付け：** ターゲット パス、データ型（テキスト、数値、日付、ブール値、画像）およびオプションの書式設定（日付マスク、通貨など）を定義します。

- **バインド プレビュー：** サンプル データセットを使用してバインドをテストし、公開前に形式と正確性を検証します。

## &#x200B;3. 使用方法

データのバインディングは、通信が信頼できるレコードを表示したり、ユーザー入力を取得したりする必要がある場合によく使用されます。 以下に例を示します。

- **Personalization:**&#x200B;名前、住所、口座残高など、お客様の詳細情報を入力します。

- **条件付きコンテンツ：** モデル値（アクティブ顧客と非アクティブ顧客など）に基づいてセクションを表示または非表示にします。

- **コレクションとテーブル：**&#x200B;配列から履歴、トランザクション、または項目別リストをレンダリングします。

- **画像とメディア：** プロファイル写真、会社ロゴ、または製品画像をバインドします。

- **レビューと電子サイン：**&#x200B;事前にフォームにデータを入力し、双方向の同期で更新を許可します。

通常、作成者はプロジェクトの初期段階でFDMを選択し、デザイン中にフィールドを視覚的にマッピングし、公開前にサンプルデータを使用してテストします。

## &#x200B;4. ベストプラクティス

- **スキーマを早期に定義：** バインドする前にFDMを確定して、後で再マッピングを行わないようにします。

- **ビジュアルマッピングを使用：** ドラッグ&amp;ドロップを使用して、タイプミスやパスの不一致を防止します。

- **データ型の検証：**&#x200B;通貨、日付、または電話番号の書式を適用して、一貫性を確保します。

- **バインディングを明示的に保持：**&#x200B;各フィールドは、単一のデータノードに明確にマッピングする必要があります。

- **サンプルデータを使用してテスト：**&#x200B;一般的なケースとエッジケースの両方をプレビューします（空の値、長い配列など）。

- **双方向の同期を制限：**&#x200B;編集または承認が必要な場合にのみ使用します。

- **コレクションをモジュール化：** テンプレート行を反復可能な構造に使用します。

- **機密データの保護：** PIIまたは支払いの詳細に対するマスキング、暗号化、最小権限アクセスを適用します。

データバインディングを慎重に設定することで、デザイナーはデザインとデータの間に信頼性の高い橋を築き、コミュニケーションのオーサリングを加速し、正確性を確保し、高度にパーソナライズされたエクスペリエンスを大規模に提供します。
