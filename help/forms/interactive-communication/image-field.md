# インタラクティブ通信エディターの画像フィールドコンポーネント


## &#x200B;1. はじめに

インタラクティブ通信エディターの画像フィールドコンポーネントを使用すると、作成者は通信レイアウトに画像を挿入できます。 写真識別、文書検証、エンドユーザーへの画像の表示が不可欠なビジュアル検証などのユースケースに最適です。

**JPEG**&#x200B;や&#x200B;**PNG**&#x200B;などの一般的なフォーマットをサポートするこのコンポーネントは、画像の表示、保存、スタイル設定を定義する柔軟な設定オプションを提供します。 作成者は&#x200B;**画像フィールドに名前またはラベルを割り当てることもでき**、レイアウトの明確性と整理を強化できます。

![IC ドキュメントを検索](/help/forms/interactive-communication/assets/imagefield.png)

## &#x200B;2. プロパティ

画像フィールドコンポーネントには、いくつかの設定可能なプロパティが含まれています。

2.1. 基本フィールド

- **名前：** データモデルとルールの参照に使用する、フィールドの一意の識別子を定義します。

- **キャプション：** インターフェイス上のユーザーに表示されるラベルを表示します。

- **画像を選択：**&#x200B;作成者がこのプロパティを使用して画像をアップロードできるようにします。このプロパティは、ロゴ、ID、またはその他のビジュアル要素で一般的に使用されます。

- **画像を予約：**&#x200B;画像、左、右、上、下の配置を定義するか、ミリメートルなどの単位（20 mmなど）を使用して&#x200B;**カスタム位置**&#x200B;を指定します。

2.2. 位置

レイアウト内の画像の空間配置を制御します。

- X座標とY座標

- 高さと幅（mm）

2.3. 余白

テキストボックスの周囲の間隔を定義します。

- 上（上）

- 下（下）

- 左

- 右

2.4. 外観

アピアランス：画像フィールドのビジュアルスタイルを定義し、境界線、フラット、高架などのプリセットを選択し、塗りの色、線幅、コーナースタイル（丸みを帯びたまたはシャープ）でカスタマイズします。

2.5. プレゼンス

実行時の画像コンポーネントの表示を決定します。

- Visible

- 非表示（スペースを保持）

2.6. データ連結

画像フィールドをデータソースにリンクして、通信内の動画像コンテンツを取得および表示します。

**データ結合タイプ：**&#x200B;画像フィールドを基礎となるデータ構造またはスキーマに接続する方法を定義します。

**使用名：** データモデルまたはスキーマで指定されたフィールド名を使用して画像フィールドをバインドし、実行時に画像を動的に入力できるようにします。

**グローバルデータを使用：**&#x200B;画像フィールドを、コミュニケーション全体で共有されるグローバルデータに接続し、ビジュアルコンテンツの一貫性を確保します。

**データバインディングなし：**&#x200B;任意のバックエンドデータからフィールドを切断したままにします。静止画像が表示される場合や、UI設定で画像を純粋に制御する場合に最適です。

## &#x200B;3. 使用方法

画像フィールドは、視覚的なコンテンツの送信が必要なコンテキストで便利です。 一般的なユースケースを次に示します。

- ID プルーフのアップロード （パスポート、ライセンス）

- プロフィールまたは個人写真の提出

- サポートドキュメントの視覚的な添付

作成者は、整列のためにサブフォームまたはレイアウトコンテナ内にフィールドを配置し、カスタム検証ルールを使用して、受け入れ可能なファイルタイプとサイズを制御できます。

## &#x200B;4. ベストプラクティス

- 画像のアップロードを求める際には、明確なラベルや指示を使用しましょう。

- パフォーマンスの検証により、ファイルサイズと形式を制限。

- アクセシビリティのためにフォールバック（予約）画像を確保します。

- バックエンドとの統合の場合、フィールドを意味のあるスキーマパスにバインドします

インタラクティブ通信エディターの画像フィールドコンポーネントは、視覚的なコンテンツのアップロードを有効にすることで、フォームのインタラクティブ性を向上させる汎用性の高いコンポーネントです。 スタイル設定、検証、データのバインディングを念頭に置いて設計されたAdobe Marketo Engageは、画像ベースの送信に対して、シームレスなユーザーエクスペリエンスと効率的なデータキャプチャを実現します。
