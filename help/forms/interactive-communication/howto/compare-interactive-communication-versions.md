# インタラクティブ通信バージョンの比較

インタラクティブ通信が複数回の編集を行う場合、保存された2つのステート間で何が変化したかを正確に把握するのは難しい場合があります。 PDFのプレビュー機能では、ふたつのバージョンを並べて開くことができ、それぞれのバージョンを個別に開いたり、スクリーンショットを手動で比較したりすることなく、レイアウトシフト、コンポーネントの追加と削除、静的なコンテンツの変更を簡単に確認できます。

| 誰が | メリット |
|-----|---------|
| **作成者（インタラクティブ通信デザイナー/所有者）** | 公開前に、レビューサイクル間の編集によって予想される変更が生成されたことを確認します。 |
| **コンテンツレビュアー** | 作成者のリビジョンで、新しい問題を導入せずに以前のバージョンからのフィードバックが処理されたことを確認します。 |

>[!NOTE]
>
> この比較では、**レイアウトと静的コンテンツのみ**&#x200B;について説明します。 実行時に入力された動的フィールド値は、比較ビューではレンダリングされません。

## 開始する前に

現在のデザインと比較するインタラクティブ通信のバージョンが少なくとも1つ保存されていることを確認します。

バージョンを作成するには、**Formsとドキュメント**&#x200B;を開き、インタラクティブ通信を選択し、左側のパネルから&#x200B;**タイムライン** パネルを開き、**バージョンとして保存**&#x200B;をクリックします。 詳しい手順については、[ インタラクティブ通信エディターでのバージョン管理とコメント ](/help/forms/interactive-communication/versioning-and-commenting-in-interactive-communication-editor.md)を参照してください。

バージョンを保存した後、現在のデザインが保存されたバージョンと異なるように、エディターでインタラクティブ通信を更新します。 比較する前に、保存されたバージョンと変更された現在の状態の両方が必要です。

## インタラクティブ通信の更新

バージョンを作成したら、インタラクティブ通信エディターでインタラクティブ通信を開き、デザインを更新します。

例えば、**Bank Interactive Communication Version 1**&#x200B;を保存した後、銀行ロゴを更新し、ローンの詳細を変更し、テーブルの車両情報を変更することができます。

- テキストベースの銀行ヘッダーを銀行のロゴ画像に置き換える
- 申込参照番号、承認済みのローン金額、ローンの期間、EMI開始日など、ローンの詳細を更新します
- 車両の詳細テーブルの&#x200B;**MakeおよびModel**&#x200B;の値とディーラー名を更新します

![ インタラクティブ通信を更新](/help/forms/interactive-communication/assets/compare-ic1.png)

編集を終了したら、**保存**&#x200B;をクリックします。 現在のデザインは、保存済みのバージョンと比較する準備ができました。

## 2つのバージョンの比較

1. **Formsとドキュメント**&#x200B;に移動し、比較するインタラクティブ通信を選択します。

1. 左側のパネルから&#x200B;**タイムライン** パネルを開きます。

1. タイムラインで、比較する保存済みのバージョンを見つけます。 例えば、「**Bank Interactive Communication Version 1**」を選択し、「**このバージョンには、新しい銀行ロゴと更新された詳細**」というコメントを付けます。

1. 「**現在の**&#x200B;と比較」をクリックします。

   ![現在の](/help/forms/interactive-communication/assets/compare-ic2.png)と比較

   新しいタブが開き、両方のバージョンがPDF プレビューとして並べて表示されます。 保存されたバージョンが左側に表示され、現在のバージョンが右側に表示されます。

   ![並べて比較](/help/forms/interactive-communication/assets/comapre-ic3.png)

   両方のプレビューをスクロールして、ページごとにレイアウトとコンテンツの違いを確認します。 たとえば、比較表では、銀行のロゴ、ローンの詳細、自動車モデルやディーラー名などの車両情報の変更が強調表示されています。

## 考慮事項

- **大きなテキストブロック：** テキストの段落またはブロックが書き換えられたり並べ替えられたりした場合、両方のページがソース PDFと同じようにレンダリングされます。 インラインテキストの差分はなく、比較は表示のみであり、変更されたテキストはフラグ付けされません。

- **動的データ：**&#x200B;動的フィールド値は比較に含まれません。 各バージョンの静的レイアウトとコンテンツのみが表示されます。

## よくある質問

**一度に何個のバージョンを比較できますか？**
選択したバージョンと現在のバージョンの2つのバージョンを同時に比較できます。新しいタブを並べて開きます。

**バージョンを現在のバージョン以外のバージョンと比較できますか？**
バージョンセレクターは常に現在のバージョンと比較されます。現在のバージョン以外の2つのバージョンを比較するには、一時的に古いバージョンに戻してから、**現在のバージョンと比較**&#x200B;をもう一度使用します。

**比較ビューをダウンロードまたはエクスポートできますか？**
いいえ。横に並べて比較するのは、ビジュアルレビューツールのみです。エディターから書き出すことはできません。
