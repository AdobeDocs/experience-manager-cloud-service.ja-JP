# よくある質問

**Q：既存の XDP をインタラクティブ通信エディターに読み込むことができますか？**
はい、既存の XDP を読み込んで、開始点として使用できます。 サポートされていない機能は、読み込み処理中にハイライト表示されます。

**Q：インタラクティブ通信エディターはオンプレミスデプロイメントで使用できますか？**
いいえ。エディターは、Forms as a Cloud Serviceのデプロイメントでのみ使用できます。
