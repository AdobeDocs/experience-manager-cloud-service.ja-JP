# よくある質問

**Q：既存のXDPをインタラクティブ通信エディターに読み込むことはできますか？**
はい。既存のXDPを読み込んで開始ポイントとして使用できます。 サポートされていない機能は、読み込みプロセス中に強調表示されます。

**Q: インタラクティブ通信エディターはオンプレミスでのデプロイに使用できますか？**
いいえ。エディターは、Forms as a Cloud Serviceのデプロイメントでのみ使用できます。
