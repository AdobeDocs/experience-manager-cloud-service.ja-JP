# はじめに {#introduction}

Adobe Experience Manager Forms as a Cloud Service は、複雑なデジタルフォームを作成、管理、公開、更新しながら、送信データをバックエンドプロセス、ビジネスルール、外部データストアに保存する、クラウドネイティブのサービスとしてのプラットフォーム（PaaS）ソリューションを提供します。このサービスは常に最新で、常に利用可能で、常に学習可能です。
