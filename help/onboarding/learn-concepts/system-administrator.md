# システム管理者 {#system-administrator}

システム管理者は、組織の一部ではなく、Adobeの一部です。 このユーザーの主な職務はシステム管理者ではない場合でも、このユーザーはAdmin Consoleの管理者権限を持つことに注意するために使用されます。 これは、契約の署名後にAdobeから最初に連絡を受ける個人で、通常はAEMにアクセスし、Cloud Serviceリソースとして設定する最初の人です。 この担当者は、Adobe Admin Consoleを使用して、アクセスから権限に至るまで、ユーザーのあらゆる側面を管理します。
