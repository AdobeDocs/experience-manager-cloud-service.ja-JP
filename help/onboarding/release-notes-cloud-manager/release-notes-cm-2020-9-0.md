# Release Notes for Cloud Manager in Adobe Experience Manager as a Cloud Service 2020.9.0 {#release-notes}

このページでは、AEMのCloud ManagerのリリースノートをCloud Service2020.9.0として概要を説明しています。

## リリース日 {#release-date}

AEMのCloud ManagerのCloud Service2020.9.0のリリース日は2020年9月3日です。

## 新機能 {#whats-new-cloud-manager}

* 「コンテンツ監査」は、「エクスペリエンス監査」という名称に変更されました。
* ビルドプロセスは、3つの Maven コマンドに分けられています。
* Gitリポジトリを複製できない場合は、最大3回再試行されます。

### バグ修正 {#bug-fixes-cm}

* 「コンテンツ監査」タブで、パブリッシュドメインではなく作成者ドメインを使用してベースURLが正しく表示されませんでした。
