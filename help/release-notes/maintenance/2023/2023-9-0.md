# メンテナンスリリースノート {#maintenance-release-notes}

次の節では、2023.9.0 機能のアクティベーションに関連する Experience Manager as a Cloud Service のメンテナンスリリースに関するテクニカルリリースノートの概要を説明します。

## リリース 13420 {#release-13420}

2023年9月12日（PT）に公開されたメンテナンスリリース 13420 の継続的な改善点を以下にまとめます。 このメンテナンスリリースは、リリース 13323 に代わるものです。

2023.9.0 機能のアクティベーションでは、このメンテナンスリリースの機能がすべて提供されます。 詳しくは、[Experience Manager リリースロードマップ](https://experienceleague.adobe.com/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap.html?lang=ja)を参照してください。

### 機能強化 {#enhancements-13420}

- ASSETS-19544: プロパティによって最終変更されたAssetsが、処理をリクエストするユーザーに設定されるようになりました。

### 修正された問題 {#fixed-issues-13420}

- ASSETS-27628:Assets検索パネルのカスタマイズ時に誤った「チャネル」ノードが作成される。
- ASSETS-27539：正規表現のマッチングに制限をアップロードします。
- ASSETS-26530：統合シェルは、ユーザーを元のページに戻しません。
- ASSETS-22719：スマート切り抜きブレークポイントの命名規則で、スマート切り抜き機能が無効になります。
- ASSETS-27726: linkshare.htmlは、Googleでインデックスを作成しないでください。
- ASSETS-27791: メタデータスキーマの検証は、最初のフィールドに対してのみ行われます。
- ASSETS-25544：無効になったCDN キャッシュ無効化ボタンを修正しました。
- ASSETS-26575：画像セットを作成する際の名前の切り捨てを修正しました。
- ASSETS-26705：非DM フォルダーアセットとコンテンツフラグメントの不要な処理を修正しました。
- ASSETS-25740：下向き矢印キーを使用して、「スマート切り抜きを編集」ページで、名前と切り抜きコントロールの役割をナレーションできないスクリーンリーダーを修正しました。
- CQ-4354266：インボックス項目を開けない。
- CQ-4354347: AEMの翻訳が更新されました。
- DISP-1009: User-AgentをX-Forwarded-Hostの非ファーストヘッダートリムとして使用します。
- アクセシビリティとセキュリティに関するさまざまな修正。

### 既知の問題 {#known-issues-13420}

なし。

## リリース 13323 {#release-13323}

2023年9月1日（PT）に公開されたメンテナンスリリース 13323 の継続的な改善点を以下にまとめます。 このメンテナンスリリースは、リリース 13239 に代わるものです。

2023.9.0 機能のアクティベーションでは、このメンテナンスリリースの機能がすべて提供されます。 詳しくは、[Experience Manager リリースロードマップ](https://experienceleague.adobe.com/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap.html?lang=ja)を参照してください。

### 機能強化 {#enhancements-13323}

- GRANITE-46784: BearerAuthenticationHandlerを無効にするオプションを追加します。
- ASSETS-26713：タッチ UI外部リンクから新しいエクスペリエンス UI ダッシュボードへ – unified-shell-integrationとui タッチ最適化アップグレード済み。
- SKYOPS-63302: com.adobe.granite:com.adobe.granite.auth.samlをv1.0.54にアップグレードします。
- GRANITE-46634：イベントクライアント 1.4.0にアップグレードします。
- GRANITE-46788: Apache Commons IO 2.13.0、Commons Lang 3.13.0、Commons Code 1.16.0、Commons Compress 1.23.0にライブラリを更新します。
- GRANITE-46705：Apache Felix Http Jetty 4.1.14 を更新します。
- GRANITE-46631: Jackrabbit バージョンを2.20.11に更新します。
- SKYOPS-61895: Jackrabbit Filevault 3.7.0にアップデートします。

### 修正された問題 {#fixed-issues-13323}

- ASSETS-28461:PDFでドキュメントクラウドビューアが機能しない（13239から修正）。
- SKYOPS-63290: バケットの誤った進化を修正しました。
- SKYOPS-54607：失敗したリクエストのRatelimiter サーバーロード計算が正しくありません。
- ASSETS-27648:ContentModelITが他のバンドルから除外ファイルを読み取れませんでした。
- GRANITE-43744：認証要件とバニティパスで設定ミスがある場合、Sling Authenticatorが正しく動作しません。
- GRANITE-46419:Auth0 IdpでのAEM統合の問題。
- GRANITE-46292:AEM Cloudのアップデート後にOkta SAML設定が機能しない。
- GRANITE-47059:Granite Jetty SSL バンドルを削除します。

### 既知の問題 {#known-issues-13323}

- SITES-15622：GraphQL - 数値およびブール値のパラメーターを持つ永続的なクエリに関する問題。
- SITES-15654：GraphQL - 同じ名前の和集合とプロパティに関する問題。
- FORMS-10652：変数が変数選択ドロップダウンに入力されない。 そのため、ユーザーはワークフローステップで変数をマッピングできません。
- CQ-4353233：ワークフローを再試行し、操作が失敗すると、誤ったエラーメッセージが表示されます。
- GRANITE-45567:AEM エクスペリエンスを削除すると、誤ったエラーメッセージが表示されます。
