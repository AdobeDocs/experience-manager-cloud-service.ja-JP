# メンテナンスリリースノート {#maintenance-release-notes}

次の節では、Experience Manager as a Cloud Service の最新のメンテナンスリリースに関する技術リリースノートの概要を説明します。

## リリース 24678 {#release-24678}

2026年3月4日（PT）に公開された、メンテナンスリリース 24678 の継続的な改善点を以下にまとめます。前回のメンテナンスリリースは、リリース 24464 でした。

2026.3.0 機能のアクティベーションでは、このメンテナンスリリースの機能がすべて提供されます。詳しくは、[Experience Manager リリースロードマップ](https://experienceleague.adobe.com/ja/docs/experience-manager-release-information/aem-release-updates/update-releases-roadmap)を参照してください。

### 機能強化 {#enhancements-24678}

* FORMS-18927: AEM Forms ファイル添付コンポーネントで、カスタム MIME タイプとファイル拡張子がサポートされるようになり、様々なドキュメントタイプを添付できるようになりました。
* FORMS-18211、FORMS-22936：チェックボックスが `<fieldset>` 要素内で正しくグループ化されず、グループラベルが最初の子として `<legend>` 内にネストされない、アクセシビリティの問題が発生しました。 これは、ナビゲーションにスクリーンリーダーを使用する障がいのあるユーザーに影響を与えました。 コアコンポーネントベースのアダプティブFormsでフィールドセットと凡例のサポートが導入され、アクセシビリティのサポートが向上しました。
パネルに「フィールドセット」オプションが追加され、ユーザーがフォーム内の関連フィールドをより効果的に整理およびグループ化できるようになりました。
* FORMS-23880：コアコンポーネントでテーマエディターがサポートされるようになりました。 この機能強化により、コアコンポーネント内のテーマをより効率的にカスタマイズおよび管理でき、デザインの柔軟性とワークフローが向上します。
* FORMS-21772: Forms Management UI にバージョン管理のサポートを追加。 この機能強化により、コアコンポーネントベースと基盤コンポーネントベースのアダプティブForms、フォームフラグメント、テーマおよびバイナリ Assetsの両方のバージョンを作成および取得でき、アセット管理とバージョン管理が向上します。
* FORMS-23094：基盤コンポーネントベースのアダプティブFormsのクライアントサイド解析が追加され、大規模法人のお客様がフォームをクラウドに移行できるようになりました。 この機能強化により、以前はサポートされていなかったコードエディタールールの EcmaScript 機能がサポートされるようになり、移行プロセスがスムーズになりました。
* FORMS-23853:sling コンポーネントで reCAPTCHA を上書きするサポートが追加されました。 この機能強化により、reCAPTCHA 設定をカスタマイズして、大規模法人のお客様の柔軟性とセキュリティを向上させることができます。
* SITES-34936：ユニバーサルエディターを使用したEdge Delivery：公開用にモデル別にコンテンツフラグメントのフィルタリングを追加する。
* SITES-36203：ユニバーサルエディターを使用したEdge Delivery：複数フィールドおよび複合マルチフィールドのサポートを有効にするコード切り替えスイッチを追加します。
* SITES-37037：ユニバーサルエディターを使用したEdge Delivery: スプレッドシートのインポートを改善して、区切り文字を自動的に検出します。
* SITES-37804：ユニバーサルエディターを使用したEdge Delivery：閉じられたユーザーグループのオーサリング（アーリーアクセス）のサポートを追加。
* SITES-38990：ユニバーサルエディターを使用したEdge Delivery：パスマッピングへの除外のサポートを追加。
* SITES-39171：ユニバーサルエディターを使用したEdge Delivery：ブロックおよびブロック項目での `cq:tags` のサポートを追加。
* SITES-40042：ユニバーサルエディターを使用したEdge Delivery：新しいサイトのデフォルトとして Config サービスを設定します。
* SITES-37649:GraphQL:JCR レベルでの複数行テキストフィールドのフィルタリングをサポートします。

### セキュリティ修正 {#security-24678}

AEM as a Cloud Service では、プラットフォームのセキュリティとパフォーマンスの最適化に取り組んでいます。 このメンテナンスリリースでは、特定された 15 つの脆弱性に対処し、堅牢なシステム保護に対する取り組みを強化しています。
