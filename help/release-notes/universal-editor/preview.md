# ユニバーサルエディタープレビューリリースノート {#preview}

ユニバーサルエディターの&#x200B;**プレビューバージョン**&#x200B;のリリースノートです。 これらの機能は現在、ユニバーサルエディターの&#x200B;**プレビュー環境**&#x200B;で利用できます。 これらの機能は、2026年6月25日（PT）に一般公開される予定です。

これらの&#x200B;**プレビュー** リリースノートは、ユニバーサルエディターに対する変更が予定されていることがわかり、プレビューバージョンに切り替えて[ テストすることで、利便性として提供されます。](/help/sites-cloud/authoring/universal-editor/navigation.md#user-properties)

>[!TIP]
>
>ユニバーサルエディターの&#x200B;**現在のリリースノート**&#x200B;については、ドキュメント [ ユニバーサルエディターリリースノート ](/help/release-notes/universal-editor/current.md)を参照してください。

>[!NOTE]
>
>実際のリリースの内容とリリース日は変更される可能性があります。

## 今後の機能 {#upcoming-features}

* AEM Sites コンソールにアクセスするための新しいボタンがツールバーに追加されました。
   * これにより、以前の&#x200B;**AEM サイト管理拡張機能** [拡張機能](/help/implementing/universal-editor/extending.md)の機能がユニバーサルエディターにネイティブで追加されます。
   * `/content/experience-fragments` パスの[Sites コンソール、](/help/sites-cloud/authoring/universal-editor/authoring.md#sites-console)または[ エクスペリエンスフラグメントコンソール ](/help/sites-cloud/authoring/fragments/experience-fragments.md)で、現在のAEM ページが開きます。
   * DAM パス （`/content/dam`）の場合、このボタンは非表示になり、現在の編集可能ファイルから一意のAEM ページを決定できない場合は非表示になります。
* アクセシビリティテクノロジーで、1文字のショートカットの機能が向上しました。

## 今後の変更 {#upcoming-improvements}

* 多数のボタンにアクセス可能な名前が正しく表示されるようになりました。
* 特定のコンテンツフラグメントをピッカーで選択した後に保持する際の問題を修正しました。
