# ユニバーサルエディタープレビューリリースノート {#preview}

ユニバーサルエディターの&#x200B;**プレビューバージョン**&#x200B;のリリースノートです。 これらの機能は現在、ユニバーサルエディターの&#x200B;**プレビュー環境**&#x200B;で利用できます。 これらの機能は、2026年7月2日に一般公開される予定です。

これらの&#x200B;**プレビュー** リリースノートは、ユニバーサルエディターに対する変更が予定されていることがわかり、プレビューバージョンに切り替えて[ テストすることで、利便性として提供されます。](/help/sites-cloud/authoring/universal-editor/navigation.md#user-properties)

>[!TIP]
>
>ユニバーサルエディターの&#x200B;**現在のリリースノート**&#x200B;については、ドキュメント [ ユニバーサルエディターリリースノート ](/help/release-notes/universal-editor/current.md)を参照してください。

>[!NOTE]
>
>実際のリリースの内容とリリース日は変更される可能性があります。

## 今後の機能 {#upcoming-features}

* エディターで`preview` [ メタタグ ](/help/implementing/universal-editor/customizing.md#meta-tags)がサポートされ、エディターが[ プレビューモードで強制的に読み込まれるようになりました。](/help/sites-cloud/authoring/sites-console/previewing-content.md)
* エディターは、編集を無効にする`readonly` [ メタタグ ](/help/implementing/universal-editor/customizing.md#meta-tags)をサポートするようになりました。そのため、ユーザーは引き続き任意のコンポーネントを選択できますが、コンテンツを変更することはできません。

## 今後の変更 {#upcoming-improvements}

* 見出し階層のアクセシビリティが改善されました。
* エディターのキーボードナビゲーションを支援するために、ロケーションバーのフォーカスを適切に維持するようにアクセシビリティが改善されました。
