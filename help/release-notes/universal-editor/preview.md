# ユニバーサルエディタープレビューリリースノート {#preview}

ユニバーサルエディターの&#x200B;**プレビューバージョン**&#x200B;のリリースノートです。 これらの機能は現在、ユニバーサルエディターの&#x200B;**プレビュー環境**&#x200B;で利用できます。 これらの機能は、2026年5月14日（PT）に一般公開される予定です。

これらの&#x200B;**プレビュー** リリースノートは、ユニバーサルエディターに対する変更が予定されていることがわかり、プレビューバージョンに切り替えて[&#x200B; テストすることで、利便性として提供されます。](/help/sites-cloud/authoring/universal-editor/navigation.md#user-properties)

>[!TIP]
>
>ユニバーサルエディターの&#x200B;**現在のリリースノート**&#x200B;については、ドキュメント [&#x200B; ユニバーサルエディターリリースノート &#x200B;](/help/release-notes/universal-editor/current.md)を参照してください。

>[!NOTE]
>
>実際のリリースの内容とリリース日は変更される可能性があります。

## その他の改善点 {#other-improvements}

* カンバスパネルがデフォルトで開くようになりました。
* [拡張アイコンがSpectrum 2に移行されました。](https://developer.adobe.com/uix/docs/services/aem-universal-editor/api/header-menu/)
