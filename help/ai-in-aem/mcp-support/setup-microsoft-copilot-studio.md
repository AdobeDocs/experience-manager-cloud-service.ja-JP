# AEM MCPを使用したMicrosoft Copilot Studioの設定 {#setup-microsoft-copilot-studio}

Microsoft Copilot StudioをAEMのMCP サーバーに接続するには、次の手順に従います。

* 新しいエージェントを作成します。
* ツール セクションに移動し、**ツールを追加**&#x200B;をクリックします。
* 既存のツールを選択するか、新しいツールを作成します。
* 1つ以上のAEM MCP サーバーURLを指す新しいMCP ツールを設定します。
* エージェント間で共有または専用できる接続を確立します。
* リダイレクト時にAdobe IDを使用してログインします。
* オプションで、自動確認モードを有効にするか、すべてのツール操作に対してエンドユーザーの確認を必要とします。
* エージェントをテストする際は、最初に接続マネージャーを開いてセッションに接続を割り当ててから、**再試行**&#x200B;を押します。
