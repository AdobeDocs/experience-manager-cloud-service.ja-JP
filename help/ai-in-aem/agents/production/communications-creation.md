# コミュニケーション作成スキル {#ic-creation-skill}

インタラクティブ通信は、アカウント明細書、ポリシードキュメント、請求書、ウェルカムキット、特典通知など、ビジネス上の通信向けに設計された、パーソナライズされたデータ駆動型ドキュメントです。 ユーザーから入力を収集するフォームとは異なり、インタラクティブ通信は、受信者固有の動的なコンテンツを含む出力ドキュメントを生成します。

コミュニケーション作成スキルは、自然言語プロンプトを使用してインタラクティブなコミュニケーションを開発するように設計された、Experience Production Agent の機能です。 このスキルにより、パーソナライズされたデータ駆動型印刷用の通信が自動的に生成されます（PDF形式）。 AI アシスタントを通じてスキルを表示します。

コミュニケーション作成スキルの主なメリットには、次のようなものがあります。

* **コミュニケーション開発の高速化**：シンプルな自然言語コマンドを使用してコミュニケーションを迅速に作成し、従来の製品インターフェイスを学習する必要がなくなります。
* **一貫したオンブランド通信**：承認済みのテンプレートとスタイルを使用して、組織のブランディング、テンプレート、スタイルガイドラインに従ったコミュニケーションを作成します。
* **技術的な障壁の軽減**：ビジネスユーザーは、高度な技術や深い製品の専門知識がなくても、簡単にコミュニケーションを作成できます。

## 機能 {#capabilities}

* **プレーンテキストプロンプトを使用してパーソナライズされたコミュニケーションを作成**：要件をプレーン言語で送信することで、印刷用のコミュニケーションドキュメント（PDF形式）を作成できます。 エージェントは、自然言語の説明に基づいて、適切なドキュメント構造、レイアウトおよびデータバインディングを自動的に生成します。

* **テンプレートから作成**：承認済みの組織テンプレートを使用して、ブランドの一貫性とコンプライアンス標準を確保できます。 このエージェントは、既存のテンプレートとスタイルガイドラインを活用して、規制要件を満たすオンブランド通信を作成します。

* **既存のドキュメントをインタラクティブ通信に読み込んで変換**：既存のドキュメントをインタラクティブ通信に読み込んで変換できます。 エージェントは、アップロードされたコンテンツを分析して、フィールドを検出し、レイアウトを保持し、動的コンテンツ機能とのデータ駆動型通信を作成します。 サポートされるファイル形式には、PDF、XDP ファイル、画像（JPG、PNG）、手書きテンプレートなどがあります。


## サンプルプロンプト {#sample-prompts}

* *取引履歴テーブルと残高グラフを使用した銀行口座取引明細書の作成*
* *業績グラフとアセット分類を含む投資ポートフォリオ概要の生成*
* *支払いスケジュールと残高を含むローン明細書の作成*
* *補償範囲の詳細と保険料の内訳を記載した保険証券ドキュメントの作成*
* *ステータスタイムラインと支払い情報を含む請求概要の生成*
* *更新された条件と価格比較表を使用してポリシー更新通知を作成する*
* *使用状況グラフ、支払い詳細、請求履歴を含むユーティリティ請求書を作成する*
* *アカウントの概要とアクション項目を含むサービス通知の生成*
* *月別消費グラフとコスト分析を含んだ年別使用状況レポートの作成*
* *報酬の詳細および福利厚生の要約を含む内定レターの作成*
* *会社のポリシーおよび必要なフォームを含んだ従業員のオンボーディングドキュメントの生成*
* *公式な情報と対応の指示を含む市民通知を作成する*

## 次の手順 {#refine-with-ic-editor}

AI アシスタントまたはExperience Hubを使用して初期のコミュニケーション構造を作成した後、インタラクティブ通信エディターを使用してドキュメントの調整と拡張を行うことができます。

* **自然言語プロンプトを使用したフィールドの編集**：プロンプトを使用すると、ドキュメント内のフィールドとそのプロパティを簡単に追加、更新、削除できます。 エージェントが指示を解釈し、ドキュメントの構造とフィールド設定に対して適切な変更を加えます。

* **増分コンポーネント作成**：ビジュアルエディターを使用して、テキスト、画像、グラフ、テーブル、バーコードなどの個々のコンポーネントを追加または変更します。

* **動的データバインディング**：データソースを設定し、フィールドマッピングを設定し、パーソナライゼーション用にコンポーネントをバックエンドデータにバインドします。

* **レイアウトとデザインの洗練**：配置、書式設定、スタイル設定、マスターページを調整して、ピクセルパーフェクトな対応を実現します。

* **フラグメントの作成と管理**：一貫性を保つために、ヘッダー、フッター、免責事項、署名ブロックなど、再利用可能なドキュメントフラグメントを作成します。

* **ルールとロジックの作成**：条件付きコンテンツの定義、ビジネスルールの設定およびルールエディターを使用した検証の設定を行います。

* **マルチチャネル出力**：印刷チャネルと web チャネルの両方のバージョンを設定し、配信形式全体で最適なプレゼンテーションを確保します。

詳しくは、[&#x200B; インタラクティブ通信ドキュメント &#x200B;](/help/forms/introduction-to-interactive-communication.md) を参照してください。
